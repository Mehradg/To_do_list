# Python To-Do List Application

A simple command-line **To-Do List Manager** built with Python.  
This project demonstrates basic file handling, JSON storage, user input validation, and menu-driven application design.  
It was created as part of my Python teaching portfolio.

---

# Features

- Add new tasks  
- View task list  
- Edit existing tasks  
- Delete tasks  
- Data persistence using JSON  
- Input validation (numeric checks, priority checks, etc.)
